# GINFV25

first proejct of GINFV25
js have 2 types:

- primitive type: number, string, boolean, null, undefined, symbol
- primitive type is immutable,
- primitive type is passed by value,
- primitive type is stored in stack,
- primitive type is compared by value,

- object type: array, function, object
- object type is mutable
- object type is passed by reference
- object type is stored in heap
- object type is compared by reference
